Here is a list of the projects I'm currently working on or are in use to analyze data sets.


Expected Goals Model:

This model was written to predict the probability of each shot becoming a goal in the NHL. Included in the link is the code to do so and the link to the data.

Twitter Stat Bot:

This includes the commands to query the twitter bot I currently have running under @barloweanalytic where users are able to request season stats or moving averages of stats of both teams and players.

NHL Game Prediction Model:

Detailing the method I use to predict the win probabilities of NHL games that are posted by @barloweanalytic on a daily basis.

Freidman Abbreviation Generator:

Created a Javascript function to generate Elliote Friedman style NHL team abbreviations.
