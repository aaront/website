Postgres Privileges

by Matthew Barlowe


These rules are often known as permissions, and in Unix are referred to as read, write, and execute permisions. These roughly translate to SELECT, UPDATE, and INSERT privileges on a table in a Postgres database.

SELECT privileges allow one to read, query the data, in a table on a database. It also allows them to /copy the data to a file from the table as well. Which means while they may not be able to change any of your data they could technically steal it so if you have sensitive information this could be a problem.
